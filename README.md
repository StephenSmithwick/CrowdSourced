============

A location based crowd sourcing review server.

Project details
--------------------
Bringing the latest in computational linguistics to crowd source reviews from tweets in your area
(MongoDB, SpringSense, Ruby, Sinatra)

Dev Environment
---------------
* Ruby 1.9.3
* Bundler
* MongoDB

Setup
-----
`$ bundle install`

To Run
------
1. Start Mongo
`$ rake mongod`    
2. Start Web Application
`$ rake`
